# Vanilla-Project-2-Color-Pallete-Generator

Color Palette Generator

This project is a JavaScript-based color palette generator that creates five random hex colors and displays them dynamically on the page. Users can generate a new palette with a button click and copy any colorâ€™s hex value to the clipboard by clicking either the color block or the copy icon.

The script uses event delegation for efficient click handling and the Clipboard API to copy values. When a color is copied, visual feedback is shown by temporarily changing the copy icon to a checkmark. A new palette is also generated automatically when the page loads.

Key Features

Generates 5 random hex colors

Updates the UI dynamically

Copy hex values via color block or copy button

Visual confirmation on successful copy

Efficient event handling using a single listener

Technologies Used

Vanilla JavaScript

DOM manipulation

Clipboard API
